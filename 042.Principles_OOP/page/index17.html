Создайте абстрактный класс MusicalInstrument, представляющий музыкальный инструмент. У класса MusicalInstrument должен быть метод play, который будет воспроизводить звук инструмента. Создайте классы-наследники Guitar, Piano и Drums, которые будут представлять гитару, пианино и ударные соответственно. Реализуйте метод play для каждого класса, воспроизводя соответствующий звук инструмента. Создайте несколько объектов классов Guitar, Piano и Drums и вызовите метод play для каждого инструмента.
